## How to use
### Just the same as using scoop

```bash
  # Add this bucket / 添加个桶
  scoop bucket add scoopcn https://github.com/scoopcn/scoopcn.git

  # Install / 安装！！

  scoop install baidudisk
  scoop install tim
```
